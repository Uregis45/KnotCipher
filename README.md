# 🔐 KnotCipher – Encryption Suite

**Tie your secrets in knots. Only the right key unties them.**

## Projects

### 1. [Standalone Encrypt/Decrypt Tool](https://uregis45.github.io/KnotCipher/)
- No server, runs entirely in your browser
- AES-GCM encryption using Web Crypto API
- Use it offline, no data leaves your device

### 2. [Ghost Chat – Real-time Encrypted Chat](https://uregis45.github.io/Cipher/)
- Firebase Realtime Database for messaging
- End-to-end AES-256 encryption
- **Unique feature:** "Ghost Mode" – messages self-destruct after 5s–5min
- Classic mode also available

## Tech Stack
- Web Crypto API (PBKDF2, AES-GCM)
- Firebase Realtime Database
- HTML5/CSS3/JavaScript
- GitHub Pages for hosting

## Why It's Unique
- Server-side cannot read messages (decryption happens locally)
- Ghost mode is rare in serverless + encrypted chats
- Two complementary tools: offline encryption + online secret chat
